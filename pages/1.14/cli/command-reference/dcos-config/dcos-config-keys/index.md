# Description

The `dcos config keys` will print all the keys that can be set in a DC/OS configuration file.

# Usage

```bash
dcos config keys
```
# Options

| Name |  Description |
|---------|-------------|
| `--help, h`   |   Display usage. |
| `--quiet, q`   |   Only print config keys. |
